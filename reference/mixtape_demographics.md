# Mixtape Sessions Cereal Demographic Data

Demographic data for the Mixtape Sessions demand estimation exercises.
Contains 20 individuals per market drawn from the Current Population
Survey.

## Usage

``` r
mixtape_demographics
```

## Format

A data frame with 1880 rows and the following columns:

- market:

  Market identifier (must match mixtape_products)

- quarterly_income:

  Quarterly income in dollars
